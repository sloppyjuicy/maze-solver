# Maze Solver

[![Build Status](https://github.com/sethjackson/maze-solver/workflows/ci/badge.svg)](https://github.com/sethjackson/maze-solver)

A maze solver.


## Prerequisites

* [Git](https://git-scm.com/)
* JDK (>= 11)


## Running

    $ git clone https://github.com/sethjackson/maze-solver

    $ cd maze-solver

    $ ./gradlew run
